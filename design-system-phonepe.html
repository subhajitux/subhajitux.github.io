Problem Statement

PhonePe, a leading digital payments platform in India, had a problem with design consistency. With multiple pods and designers working on different features, the UI elements used in the designs from each pod looked different. This made the app look cluttered and unprofessional.

Solution

To solve this problem, We decided to rebuild its existing design system and come up with guidelines for UI design. The new design system consists of foundational elements, UI components, and detailed guidelines on their usage.

Color Handling in Light, Dark, and Dark Purple Modes

The PhonePe design system includes colors for both light and dark modes, as well as the current dark purple mode of PhonePe. The dark purple mode will be eventually replaced with the dark mode, but it is still included in the design system to support older devices and users who have not yet switched to the dark mode.

Implementation

Once the new design system was complete, we began working with the product teams to implement it in the PhonePe app. We provided training to the designers and developers on how to use the new design system and follow the guidelines.

Results

The new design system has had a significant impact on the consistency of the UI design in the PhonePe app. The app now looks more polished and professional, and it is easier for users to navigate.

Benefits

‚û°Ô∏è Consistency: The design system ensures that the UI elements used in the PhonePe app are consistent across all screens and features. This makes the app easier for users to learn and use.

‚û°Ô∏è Efficiency: The design system saves designers and developers time by providing them with pre-built UI components and guidelines. This allows them to focus on creating new features and improving the user experience.

‚û°Ô∏è Scalability: The design system is scalable, meaning that it can be easily adapted to meet the needs of a growing product.

Conclusion

PhonePe's new design system is a success story. It has helped to improve the consistency and efficiency of UI design at PhonePe, and it has made the app easier for designers to learn and use. The design system is also scalable, meaning that it can be easily adapted to meet the needs of a growing product.
